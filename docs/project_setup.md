## Prerequisites

Before you begin, make sure you have the following installed:

- **Python 3.10 or higher** 
- **pip** – Comes with Python, or install via `python -m ensurepip`
- **virtualenv** 
- **Django 5.0+**
- **PostgreSQL 13+**
- **Redis (recommended)**
- **Git** – For cloning the repository
- **Node.js 16+**
